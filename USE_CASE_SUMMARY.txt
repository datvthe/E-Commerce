═══════════════════════════════════════════════════════════════
  🎨 CÁCH SỬ DỤNG FILE PLANTUML
═══════════════════════════════════════════════════════════════

OPTION 1: Online
  1. Mở: https://www.plantuml.com/plantuml/
  2. Copy code từ use_case_diagrams.plantuml
  3. Paste vào
  4. Xem diagram tự động generate
  5. Download ảnh PNG/SVG

OPTION 2: VS Code
  1. Cài extension: "PlantUML"
  2. Mở file use_case_diagrams.plantuml
  3. Press Alt+D (preview diagram)
  4. Export PNG/SVG

OPTION 3: IntelliJ/PyCharm
  1. Cài plugin: "PlantUML Integration"
  2. Mở file .plantuml
  3. View diagram trong editor

BƯỚC 1: Đọc hướng dẫn

  → Visual Paradigm
  → PlantUML (nếu thích code)

  → Bắt đầu với Guest (đơn giản nhất)
  → Sau đó Customer
  → Rồi Seller
  → Cuối cùng Admin và Moderator

BƯỚC 4: Vẽ diagram tổng quan
  → Tất cả actors trong 1 diagram
  → Nhóm use cases theo chức năng
  → Thể hiện quan hệ kế thừa

BƯỚC 5: Mô tả chi tiết
  → Chọn 5-10 use case quan trọng
  → Viết mô tả chi tiết theo template
  → Bao gồm: Main flow, Alternative flow, Exception flow
